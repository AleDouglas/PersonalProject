##### Relatório #####

Os valores do relatório são gerados a partir do mês atual salvo no sistema. Para o relatório ir além do mês do próprio sistema, basta apenas acessar a área de Projeções no menu principal.

# Divida total: R${{divida_total}}0 || CLIQUE AQUI PARA ADICIONAR NOVOS VALORES

Junção de todas as dívidas do mês atual. Ela usa de referência o inicio da dívida para verificar se ainda estará pendente no próximo mês.

# Dívida mensal Em: R${{divida_mensal}}0 || CLIQUE AQUI PARA ADICIONAR NOVOS VALORES

# Valor pago nesse mês: R${{pagamento_mes}}0 || CLIQUE AQUI PARA ADICIONAR NOVOS VALORES

# Saldo do mês: R${{ganhos_mes}}0 || CLIQUE AQUI PARA ADICIONAR NOVOS VALORES

Referente a todos os ganhos do mês, caso surja novos valores no decorrer do mês esse valor é alterado também.

# Saldo após pagamentos: R${{ganhos_mes}}0 - R${{pagamento_mes}}0 = R${{saldo_apos}}0

O saldo após pagamentos é o dinheiro livre restante para o mês. Lembrando que esse saldo abrange apenas os pagamentos feitos até agora, caso surja novos pagamentos esse valor é automaticamente reduzido.
